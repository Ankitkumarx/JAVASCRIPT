# JavaScript Practice Folder

This folder contains JavaScript practice files for learning fundamental concepts such as:

- Variables and data types (`variable.js`)
- Operators and type coercion (`operators.js`, `Typecoersion.js`)
- Control flow and loops (`loop.js`, `if_else.js`)

## Files

- `variable.js` - demonstrates variable declaration, initialization, and input handling with Node.js readline.
- `operators.js` - demonstrates arithmetic, assignment, comparison, logical, and type-reassignment examples.
- `Typecoersion.js` - examples of JavaScript type coercion behavior.
- `loop.js` - examples of loop structures and iteration.
- `if_else.js` - examples of conditional statements.

## Run a file

Use Node.js to run any example file:

```bash
node variable.js
```

Replace `variable.js` with the file you want to execute.
